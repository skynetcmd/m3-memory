# Examples

This folder contains example integrations and reference implementations built on M3 Memory.

| Folder | Description |
|--------|-------------|
| `homelab-dashboard/` | Homelab monitoring dashboard with M3 Memory backend |
| `mac-agent/` | macOS agent integration example |
| `sandbox-openclaw/` | OpenClaw sandbox configuration |

These are reference/dev workspaces. For the main quick start, see the [root README](../README.md).
