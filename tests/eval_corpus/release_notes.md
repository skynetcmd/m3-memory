# Release notes

## v2.4.0 (2026-03-15)

- Added support for the new MX-9 mounting bracket on medium widgets.
- Increased default cable length from 2.0 to 2.5 meters.
- Fixed a bug where small widgets would overheat above 45 degrees.

## v2.3.0 (2026-01-08)

- Renamed "Tiny" widgets to "Small" for consistency.
- New graphite finish available exclusively on large widgets.
- Deprecated the legacy MX-7 bracket.

## v2.2.0 (2025-11-22)

- Introduced the medium widget size at 4.7 kg.
- Maximum duty cycle for large widgets raised to 100 percent.
- Migrated all internal docs to the new format.
